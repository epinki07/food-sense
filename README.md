## El proyecto Food Sense

Food Sense es un sistema IoT de monitoreo remoto de refrigeradores con camaras, sensores y alertas en tiempo real. Fue desarrollado en equipo durante la competencia Invent for the Planet 2026 donde obtuvo **3er lugar**.

**Concepto:**
- Monitoreo de temperatura y humedad en refrigeradores
- Deteccion visual del estado de alimentos
- Alertas automaticas ante fallas de refrigeracion
- Dashboard web para visualizar multiples unidades

## Landing page (sqad-landing/)

Pagina web del proyecto que presenta la propuesta de valor de Food Sense.

```bash
cd sqad-landing
npm install
npm start
```
